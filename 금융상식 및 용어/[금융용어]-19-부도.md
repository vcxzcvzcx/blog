# 부도

| 비교 |          |              |                    |                |
| :--: | :------: | :----------: | :----------------: | :------------: |
| 인간 |   출생   |     유언     |        사망        |    사망선고    |
| 법인 | 법인등기 | 청산 or 파산 | 해산(법인활동종료) | 법인등기부폐쇄 |

> [!TIp]
>
> * 청산: 해산시 재산정리 (자산 > 부채)
> * 파산: 해산시 재산정리 (자산 < 부채)

> [!caution]
>
> * `폐업`은 기업이 죽진 않는 것이다. 
> * 이는 `사업자 등록말소`라고 하는데, 법인이 신시장 좋은 것 같아서 법인사업자 등록 후 폐업 하는 것
