# 직접세 vs 간접세

| 구분   |                                                              | 예시           |
| ------ | ------------------------------------------------------------ | -------------- |
| 직접세 | 납세자(세금 낼 의무가 있는 사람)와 담세자(실제 세금 내는 사람)가 일치하는 조세 | 소득세, 법인세 |
| 간접세 | 납세자와 담세자가 불일치 하는 조세                           | 부가가치세     |

> [!caution]
>
> * 직접세가 조세저항이 많다? --> 내 주머니에서 목돈 나가니 뺏기는 기분이여서 싫은 것
